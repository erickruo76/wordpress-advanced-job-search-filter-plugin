== Description ==
This plugin adds a job search form with advanced filtering options including category, location, and job type.

== Installation ==
1. Upload `job-search-plugin.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[job_search_form]` and `[custom_archive_title]` shortcodes in posts, pages, or templates.

== Shortcodes ==
- `[job_search_form]`: Adds the job search form.
- `[custom_archive_title]`: Displays the custom archive title.
